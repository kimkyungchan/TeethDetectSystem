## Function
  * 충치를 검진하는 YOLO 모델 학습
  * 사진을 통해 충치를 검진 기능 구현
  * 외장 카메라를 이용한 충치 검진 기능 구현
  * Google Map, Open Api(치과 위치)를 사용한 주변 치과 찾기 기능 구현
  * 회원가입/로그인 및 회원 정보 관리 기능 구현
